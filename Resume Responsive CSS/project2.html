Project Title: Monster University Parking System

Project Description

This project is a command-line parking monitoring system for Monster University. It tracks multiple parking garages with different capacities, keeps a list of registered vehicles and owners, and manages vehicles entering/exiting garages. The system supports real-time updates like parking, relocating, resizing garages, removing garages, searching for an owner by license plate, and generating utilization reports.

Technologies Used

  • C
  • Github
  • Dynamic Memory (malloc, calloc, realloc, free)
  • Command-line Input/Output

Skills Learned

Throughout the development of this project, I strengthened my understanding of data structures in C and memory management. I also improved at writing modular functions, building a clean command-driven program that can manages multiple related structures.

  • Dynamic memory management in C
  • Designing and working with structs
  • Managing arrays of pointers
  • Debugging memory leaks and pointer issues
